 # 📄 Resume Fit Analyzer

I built this project to check how well a resume matches a job description.

Most of the time, we apply to jobs without really knowing if our resume actually fits.  
This tool gives a quick idea of that.

---

## 🧠 What it does

- Takes resume and job description as input  
- Calculates a match score  
- Shows missing keywords  
- Gives simple suggestions  

---

## 💡 Why I made this

I wanted something simple that tells me if I’m even a good fit before applying.  
Instead of guessing, this gives a basic data-driven answer.

---

## ⚙️ Tech used

- Python  
- Scikit-learn  
- Streamlit  

---

## ▶️ How to run

Install dependencies:

pip install scikit-learn streamlit  

Run the app:

streamlit run app.py  

---

## 📂 Note

You can paste any resume and job description to test it.
